**279. 完全平方数**

## 题目描述
给你一个整数 n ，返回 和为 n 的完全平方数的最少数量 。

完全平方数 是一个整数，其值等于另一个整数的平方；换句话说，其值等于一个整数自乘的积。例如，1、4、9 和 16 都是完全平方数，而 3 和 11 不是。

 

**示例 1：**
```
输入：n = 12
输出：3 
解释：12 = 4 + 4 + 4
```

**示例 2：**
```
输入：n = 13
输出：2
解释：13 = 4 + 9
```

**提示：**
```
1 <= n <= 104
```
### 1. 明确状态定义 (dp 数组的含义)
我们需要一个数组 `dp`，其中 **`dp[i]` 表示和为 `i` 的完全平方数的最少数量**。
我们的最终目标是求出 `dp[n]`。

### 2. 推导状态转移方程
对于任意一个数字 `i`，它是怎么由之前的状态推导过来的呢？
假设我们在凑成数字 `i` 的时候，选择的最后一个完全平方数是 `j*j`（显然，`j*j` 必须小于等于 `i`）。
那么，除了这个 `j*j` 之外，剩下的部分也就是 `i - j*j`。而凑成 `i - j*j` 所需的最少平方数数量，我们在之前肯定已经算过了，也就是 `dp[i - j*j]`。

所以，凑成 `i` 的数量就是 `dp[i - j*j] + 1`（这个 `+1` 就是指我们刚刚选的那个 `j*j`）。
因为我们要找的是**最少数量**，所以我们需要遍历所有可能的 `j`（即 `1, 2, 3...` 直到 `j*j <= i`），并在所有的 `dp[i - j*j] + 1` 中取最小值。

> **状态转移方程：** `dp[i] = min(dp[i], dp[i - j*j] + 1)`，其中 `1 <= j*j <= i`。

### 3. 初始化 dp 数组
* **`dp[0] = 0`**：凑成和为 0 的平方数数量显然是 0。虽然题目是从 1 开始的，但 `dp[0]` 是整个递推的基础。例如，凑成数字 4 时，如果选了 4 这个平方数，剩下的就是 `dp[4-4] = dp[0]`，此时总数量为 `dp[0] + 1 = 1`，这是符合逻辑的。
* **其他位置 `dp[i]` 初始化为无穷大**（在代码中也可以用 `i` 来初始化，因为最坏的情况就是由 `i` 个 1 组成）：因为我们要求最小值，如果初始化为 0，在后续执行 `min()` 操作时就会被 0 覆盖，导致错误。

### 4. 确定遍历顺序
由于后面的状态依赖于前面的状态（`dp[i]` 依赖于较小的 `dp[i - j*j]`），所以我们需要**从前往后**遍历 `i`。

---

### Python 代码实现

下面是根据上述分析写出的代码，非常直观：

```python
class Solution:
    def numSquares(self, n: int) -> int:
        # 初始化 dp 数组，长度为 n+1，初始值设为一个不可能的大数（这里用无穷大）
        # dp[i] 代表和为 i 的完全平方数的最少数量
        dp = [float('inf')] * (n + 1)
        
        # 边界条件：和为 0 需要 0 个数字
        dp[0] = 0
        
        # 遍历背包容量：也就是要求的目标和 i
        for i in range(1, n + 1):
            # 遍历物品：也就是可选的完全平方数 j*j
            j = 1
            while j * j <= i:
                # 状态转移：当前状态等于当前状态与选了 j*j 后的状态的较小值
                dp[i] = min(dp[i], dp[i - j * j] + 1)
                j += 1
                
        return dp[n]
```

### 复杂度分析
* **时间复杂度**：O(n * sqrt(n))。外层循环执行 `n` 次，内层循环最多执行 `sqrt(i)` 次，综合起来时间复杂度大约是 O(n * sqrt(n))。
* **空间复杂度**：O(n)。我们需要一个长度为 `n + 1` 的数组 `dp` 来存储中间状态。
